## 2026-04-09 - Optimize large city list rendering\n**Learning:** The application renders a large list of over 1450+ cities directly into the DOM inside a scrollable container in both `src/pages/Index.tsx` and `src/pages/BuildBet.tsx`, causing significant DOM bloat and performance bottlenecks (especially noticeable when typing in the search box). \n**Action:** Implemented scroll-based pagination in these components to prevent massive DOM bloat without removing the user's ability to browse the full list. Additionally, using `React.memo` requires explicitly setting the correct type signature (e.g., `React.memo<Props>(...)`) to prevent TypeScript errors.
